# Hygate Network
Collection of assets/mods/plugins/prefabs about creating custom portals in Hytale.

* [HygateAuto](https://github.com/HygateNetwork/HygateAuto)
  &minus; Plugin that generates custom portals based on available worlds (not instances) in the game.
* [HygateManual](https://github.com/HygateNetwork/HygateManual)
  &minus; Simple, non-code asset pack to define custom portals by hand.
* [HygatePrefabs](https://github.com/HygateNetwork/HygatePrefabs)
  &minus; Prefabs like portal gateways and buildings related to Hygate for Hytale.
